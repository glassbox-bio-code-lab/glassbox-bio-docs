# Choose Your Surface

Use this page when you know your goal but not yet the right docs section.

## I need to prepare or validate inputs before execution

Start with **PreFlight UI** and the input-contract pages that support it.

Best first pages:

- [Supported Inputs](../preflight-ui/validation-system/supported-inputs.md)
- [Validation Rules](../preflight-ui/validation-system/validation-rules.md)
- [Prepare Inputs](../computational-safety-diligence/prepare-inputs.md)

## I need to run the core pipeline and interpret its outputs

Start with **Computational Safety Diligence**.

Best first pages:

- [Create and Configure a Run](../computational-safety-diligence/create-and-configure-a-run.md)
- [Outputs Overview](../computational-safety-diligence/outputs/overview.md)
- [Interpret Results](../computational-safety-diligence/interpret-results.md)

## I need to deploy, operate, or troubleshoot the runtime bundle

Start with **Deployment and Operations**.

Best first pages:

- [Kubernetes and Helm](../deployment-and-operations/kubernetes-and-helm.md)
- [Configuration](../deployment-and-operations/configuration.md)
- [Troubleshooting](../deployment-and-operations/troubleshooting.md)

## I need exact file names, config keys, or schema families

Start with **Reference**.

Best first pages:

- [Output File Reference](../reference/output-file-reference.md)
- [Config Reference](../reference/config-reference.md)
- [Schemas](../reference/schemas.md)

## I need optional product extensions

Start with **Add-Ons**.

Use the add-on overview and workflow pages to find the extension that matches your task.
